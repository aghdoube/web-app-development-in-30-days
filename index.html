Cheatsheets & Shorthands

HTML Cheatsheet

  • DOCTYPE: Declares the document type.
  • <html>: Root element of an HTML page.
  • <head>: Contains meta-information about the document.
  • <body>: Contains the content of the document.

CSS Cheatsheet

  • Selectors: Target HTML elements (e.g., body, .class, #id).
  • Properties: Define styles (e.g., color, background, font-size).
  • Box Model: Margin, Border, Padding, Content.

JavaScript Cheatsheet

  • Variables: Use let, const, or var to declare.
  • Functions: Use function declarations or arrow functions.
  • Events: Use addEventListener for handling events.
